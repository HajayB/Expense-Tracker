Everything you need

A complete personal finance toolkit in one lightweight app.

📋

Transaction Log

Add, edit, and delete income or expense entries. Paginated table with live search, type filter, and date range.

🗂️

Custom Categories

Create your own categories with a name, emoji icon, and type (income / expense / both). Full CRUD management.

📊

Visual Summary

Doughnut chart for category breakdown, bar chart for totals, and a 6-month trend line. Filter by date range.

🏠

Dashboard Overview

At-a-glance balance, total income, and total expenses with your 4 most recent transactions.

🌙

Dark Mode

Full dark mode support with flash-free initialization. Preference persists across sessions via localStorage.

🔐

Secure Auth

JWT authentication with MongoDB-backed token blacklist on logout. Rate-limited auth routes and Zod input validation.

Tech Stack

Built with modern, lightweight tools — no frontend framework needed.

Backend

🟢 Node.js ⚡ Express 🍃 MongoDB 🔗 Mongoose 🔑 JWT ✅ Zod 🛡️ express-rate-limit ♻️ nodemon

Frontend

🌐 Vanilla JS 🎨 Tailwind CSS v3 📈 Chart.js 🔤 Poppins 🍞 Toast Notifications 💀 Skeleton Loaders

API Reference

All endpoints require a Bearer <token> header except auth routes.

🔐 Auth

POST/api/users/signup
POST/api/users/login
POST/api/users/logout
GET/api/users/profile

💳 Transactions

GET/api/transactions?type&search&startDate&endDate&page&limit
POST/api/transactions
PUT/api/transactions/:id
DELETE/api/transactions/:id
GET/api/transactions/summary?startDate&endDate

🗂️ Categories

GET/api/categories
POST/api/categories
PUT/api/categories/:id
DELETE/api/categories/:id

📦 Summary Response Shape

{
  balance: Number,
  income: Number,
  expenses: Number,
  totalTransactions: Number,
  expensesByCategory: { [key]: Number },
  monthly: [
    { label: "Jan 25", income: N, expenses: N }
  ]
}
